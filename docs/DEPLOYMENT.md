# Publicação

Primeira opção planejada: Vercel para a aplicação Next.js, Firebase para dados/autenticação e Cloudinary para imagens. Domínio e DNS poderão ficar na Cloudflare.

Esta decisão é provisória. O domínio, repositórios e serviços não dependem de APIs proprietárias da Vercel. Não há publicação autorizada nem configuração de produção nesta entrega.

Antes de publicar, configurar `SITE_URL` com a origem pública completa, por exemplo `https://cardapio.pizzaexpress.com.br`. Em desenvolvimento, metadata, sitemap e robots utilizam `http://localhost:3000`.
